part_packages.xml

The part_packages.xml configuration file defines packages of parts that can be used for initial settlement supplies and resupply missions.

The "part-package-list" element is a list of all part packages used in the simulation.

The "part-package" element defines an individual part package. It has a unique "name" attribute. It contains a list of "part" elements representing parts contained in the package.

Each part element must have a "type" attribute that matches a part type (see conf/parts.xml). The "number" attribute is the number of this type of part contained in the part package.
