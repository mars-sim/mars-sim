Keyboard Shortcuts

(A). Alt + T : Display shortcuts for all Tool Windows.
  • F1 : Mars Navigator
  • F2 : Settlement Map
  • F3 : Entity Browser
  • F4 : Monitor Tool
  • F5 : Event Viewer
  • F6 : Commander Dashboard
  • F7 : Console Window
  • F8 : Create a New Mission
  • F7 : Create a New Transport Item
  • Ctrl + F1 : Orbit Viewer
  • Ctrl + F2 : Time Tool
  • Ctrl + F3 : Search Tool

Note : see User Interface for more details on each Tool.


(B). Alt + H : Display shortcuts for viewing documents.
  • Ctrl + O : Open the About page in the Help Browser.
  • Ctrl + C : Open the Change Log in the Help Browser.
  • Ctrl + V : Check New Version of mars-sim.
  • Ctrl + T : Open the Tutorial in the Help Browser.
  • Ctrl + G : Open the User Guide in the Help Browser.

(C). Alt + F : Display shortcuts for file-related operations.
  • Ctrl + S : Save the simulation with the default filename 'default.sim'.
  • Ctrl + A : Save As the simulation with the filename of your choice.
  • Ctrl + X : Exit the Simulation.

(D). Alt + S : Display shortcuts for other settings.
  • Ctrl + U : Toggle on and off the Unit Bar.
  • Ctrl + B : Toggle on and off the Tool Bar.
  • None : Check the use of Desktop Browser vs. Inline Help Browser.
  • None : Open Look and Feel menu.
  • None : Open the floating bar of the background music and sound effect volume.
  • None : Choose Notification Settings.
  • None : Choose Docking UI.
  • None : Choose Window UI.
