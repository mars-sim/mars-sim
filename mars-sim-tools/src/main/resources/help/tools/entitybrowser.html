Entity Browser

The Entity Browser provides a tool for viewing and searching for entities in the simulation. It provides a tree view of all entities in the simulation, organized by type. The user can expand and collapse the tree to view entities of interest. The user can also search for entities by name using the search bar at the top of the Entity Browser.

Clicking on an entity in the tree will display detailed information about that entity in the panel on the right. The information displayed will depend on the type of entity selected, but may include properties such as name, type, location, status, and other relevant details.

The Entity Browser is a useful tool for quickly finding and viewing information about specific entities in the simulation, especially when there are many entities present. It can be used to monitor the status of various entities, track their locations, and gain insights into the overall state of the simulation.
