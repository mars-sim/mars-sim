Event Viewer

The Event Viewer provides a tool for viewing and searching for events in the simulation. It provides a list view of all events in the simulation, organized by type. The user can expand and collapse the list to view events of interest. The user can also filter events by their category at the top of the Event Viewer.

Clicking on an event in the list will display detailed information in an expandable panel. The information displayed will depend on the type of event selected, but may include properties such as name, type, location, status, and other relevant details. Events can be acknowledged in the panel so that they are removed from the list.

The Event Viewer is a useful tool for quickly finding and viewing information about specific events in the simulation, especially when there are many events present. It can be used to monitor the status of various events, track their occurrences, and gain insights into the overall state of the simulation.
