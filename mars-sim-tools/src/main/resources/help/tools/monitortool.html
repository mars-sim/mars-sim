Monitor Tool

The Monitor tool displays information in the format of a table layout for a total of nine different types of data.

These numerous Data Tables are accessible by clicking on its tab at the bottom of the table layout in the Monitor Tool.

  • Bots
  • Crops
  • Food
  • Trade Goods
  • Missions
  • People
  • Settlements
  • Vehicles

The Monitor tool has the following Statistics Tools built-in. They are shown as buttons on the top of the table.

  • Pie Chart - Creates a tab with a pie chart of one column from the table.
  • Bar Chart - Creates a tab with a bar chart of multiple columns from the table.
  • Remove Tab - Removes the currently selected custom tab.
  • Center Map On Selected Unit - Centers the map and globe in the Mars Navigator tool on the selected unit.
  • Show Details Dialog - Opens a information window on the selected unit.
  • Change The Displayed Columns - Lets you modify what information is displayed in the Monitor tool.
