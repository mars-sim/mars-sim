User Interface

The Mars Simulation Project provides the following visual tools and elements at your disposal. It can present content in 2 styles:

  • Internal Window Style - Multiple internal windows can be opened and managed within the main application window.
  • Docking Window Style - Tools and windows can be detached and placed within regions of the window. This is akin to an IDE.
The Settings menu allows you to switch between the two styles. The default style is Internal Window Style. You can also set the default style in the configuration file (msp.properties) by changing the value of "ui.style" to "internal" or "docking".

Both styles have the same visual elements and tools, but their arrangement and interaction differ.

Toolbar and Menubar Tool Bar

This provides access to various tools and functions, such as file operations, simulation controls, and utility tools. The toolbar can be toggled on/off via Ctrl + B.

Entity Info Window on the Main Tab

Each Entity within the simulation has a dedicated info window that displays its attributes and status. The info window can be accessed by clicking on the Entity's icon on the map, the Entity button in info panels or by selecting the Entity from the Entity Browser.
  • Person Info Window
  • Bot Info Window
  • Vehicle Info Window
  • Settlement Info Window
  • Equipment Info Window
  • Scientific Study Info Window
  • Mission Info Window
  • Resupply Info Window

Entity Bar providing shortcuts to various entities info windows.

This is only available in the Internal Window Style. It provides quick access to various entities info windows, such as Person, Bot, Vehicle, Settlement, Equipment, Scientific Study, Mission and Resupply info windows. The Entity Bar can be toggled on/off via Ctrl + E.
