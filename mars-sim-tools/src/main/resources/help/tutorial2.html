Tutorial - Step 2 : The Main Simulation Window

The Main Simulation Window is the overall window area upon which the player may query how the simulation is doing.

Besides the main window, you should see a terminal window or the Console Window, which houses the MarsLink A.I. that provides text interactions with players.

In window mode, you will see a row of icons right below title bar.

The desktop area hosts the content of any tools you may have opened.

At the start of a brand new simulation, it defaults to opening the Help Browser.

If you choose the default scenario, all settlements that were pre-defined in Simulation Configuration Editor in the simulation will appear. You will be able to see the quick status of each of them by clicking the 'Settlements' tab in the Monitor Tool.
