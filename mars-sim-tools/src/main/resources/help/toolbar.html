Tool Bar

The buttons on the tool bar perform the following actions:

1. File Buttons

  • New... - Creates and starts a new simulation. Opens the Simulation Configuration Editor.
  • Open... - Loads a simulation from a save file.
  • Save - Saves the current simulation to the default save file.
  • Save As... - Saves the current simulation to a save file.

2. Tool Buttons

  • Mars Navigator - Opens the Mars Navigator tool.
  • Search Tool - Opens the Search tool.
  • Time Tool - Opens the Time tool.
  • Monitor Tool - Opens the Monitor tool.
  • Settlement Map Tool - Opens the Settlement Map tool.
  • Commander's Dashboard - Opens the Commander Tool.
  • Event Viewer - Opens the Event Viewer.
  • Entity Browser - Opens the Entity Browser.
  • Console Tool - Opens the Console Tool.
  • Resupply Tool- Opens the Resupply Wizard.
  • Mission Tool - Opens the Mission Wizard.
  • Audio Control - Opens the Audio Control.

3. Other Buttons

  • Timestamp Bar displays the current Earth and Martian date and time.

  • Calendar button opens up a Martian calendar.

  • Orbit Viewer button opens up a simple solar system view of each planet and control buttons for updating planetary orbits.

  • Help - button Opens the help browser.

Note: The whole tool bar can be toggled on/off via Ctrl + B.

4. Menu Bar

All the above are also available in the MenuBar. The Menu Bar also provides access to the Setting:
  • Change the Look and Feel following the available FormDev LAF styles.
  • Change the window style buit this need a restart.
