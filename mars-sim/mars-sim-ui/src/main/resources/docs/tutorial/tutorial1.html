Tutorial - Step 1: Introduction

Welcome to the in-simulation tutorial of Mars Simulation Project.

Through these steps you will become familiar with the basic tools and viewers and learn how to overview and interact with the simulation. Expect it to take around 10 minutes.

The tutorial is supplemented by a more section-oriented guide, also accessible from the Help menu.

You can close this tutorial window at any time.

When you opened Mars Simulation Project for the first time, you were likely presented with two things that might seem confusing. First you were told that the 'default simulation did not load', and second you were asked to create a new simulation. Assuming you just clicked 'Create New Simulation' on the latter, you should now have been taken to the main simulation window, which has 'Mars Simulation Project' and a version number in its title.
