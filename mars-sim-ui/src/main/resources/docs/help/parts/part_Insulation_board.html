Part : "Insulation board"


1. Mass Per Unit : 1.0kg

2. Description :

  • Extruded polystyrene (XPS) and expanded polystyrene insulation (EPS) contain millions of air pockets trapped within the foam. Because the polystyrene itself is also highly resistant to heat, the result is a great thermal insulator which can give excellent R-ratings. In addition to this, XPS is often lined with aluminium foil
