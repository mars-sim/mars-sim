What's New in v3.4.0

11 Nov 2021

New Features

  1. Implement airlock reservation system for EVA egress.
  2. Add new Computation building function, new Computer Science subject, new Computer Scientist job, new Computing specialist and Chief of Computing role, and new Computing skill.
  3. Make a person's carrying capacity age-dependent.
  4. Enable lightweight inventory for all units.
  5. Add base mass to all buildings, in preparation for calculating overall rocket payload mass.
  6. Add building general info such as dimensions and mass.
  7. Arriving future Settlements are configured from a Scenario.
  8. Scenarios can be exported and shared.
  9. Add garment.

Fixes

  1. Consider loading/Unloading edge cases for Trading/Delivery.
  2. Resolve stalled delivery negotiation.
  3. Exploration Mission revisits existing sites until they reach an evaluation ready for Mining.
  4. Ground vehicle maintenance no longer stalls when EVA is aborted.
