Menu Bar

The menu bar which is always at the top of the mars-sim window has the following commands:

File

  • New... - Creates and starts a new simulation. Opens the Simulation Configuration Editor dialog.
  • Save - Saves the current simulation to the default save file.
  • Save As... - Saves the current simulation to a save file with supplied name.
  • Exit - This closes the Mars Simulation Project. Your progress is automatically saved.

Tools

The Tools menu handles the visible tool windows for interacting with the simulation.

  • Mars Navigator - Checkbox opens or closes the Mars Navigator tool.
  • Search Tool - Checkbox opens or closes the Search tool.
  • Time Tool - Checkbox opens or closes the Time tool.
  • Monitor Tool - Checkbox opens or closes the Monitor tool.
  • Settlement Map Tool - Checkbox opens or closes the Settlement Map tool.
  • Mission Tool - Checkbox opens or closes the Mission tool.
  • Science Tool - Checkbox opens or closes the Science tool.
  • Resupply Tool - Checkbox opens or closes the Resupply tool.

Settings

The Settings menu lets you set simulation-wide preferences like sound volume or the display of certain user interface items.

  • Full Screen Mode - This toggles the display of mars-sim window between the full screen mode and the window mode.
  • Skin Theme - Choose an available theme, namely, 'Standard' and 'Snow'. for all mars-sim window frames and panels
  • Volume Up - Nudges the overall sound volume UP by a tenth.
  • Volume Down - Nudges the overall sound volume DOWN by a tenth.
  • Mute - Turns off or on all sound output. Volume changes will not be audible before mute is removed again.

Help

The Help menu offers some items that present information to help you using Mars Simulation Project better. They open in the same window and there are some simple back, forward and home navigation buttons to help you move around.

  • About - Opens a window with information about mars-sim, including credits and license info.
  • Tutorial - Shows a step-by-step tutorial for new users.
  • User Guide - Opens this User Guide window.
