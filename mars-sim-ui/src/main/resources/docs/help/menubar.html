Menu Bar

The menu bar has the following commands:

File

  • New... - Creates and starts a new simulation. Opens the Simulation Configuration Editor dialog.
  • Open... - Loads a simulation from a save file chosen by the user.
  • Save - Saves the current simulation to the default save file.
  • Save As... - Saves the current simulation to a save file with supplied name.
  • Exit - Closes the Mars Simulation Project. Your progress is automatically saved.

Tools

  • Mars Navigator - Checkbox opens or closes the Mars Navigator tool.
  • Search Tool - Checkbox opens or closes the Search tool.
  • Time Tool - Checkbox opens or closes the Time tool.
  • Monitor Tool - Checkbox opens or closes the Monitor tool.
  • Preferences Tool - Checkbox opens or closes the Preferences tool.
  • Settlement Map Tool - Checkbox opens or closes the Settlement Map tool.
  • Mission Tool - Checkbox opens or closes the Mission tool.
  • Science Tool - Checkbox opens or closes the Science tool.
  • Resupply Tool - Checkbox opens or closes the Resupply tool.

Help

The Help menu offers some items that present information to help you using Mars Simulation Project better. They open in the same window and there are some simple back, forward and home navigation buttons to help you move around.

  • About - Opens a window with information about the project, including credits and license info.
  • Tutorial - Shows a step-by-step tutorial for first time users.
  • User Guide - Opens this User Guide window.
