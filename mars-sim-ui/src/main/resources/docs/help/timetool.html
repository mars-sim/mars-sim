Timekeeping

Timekeeping on Mars is vital for success of running a Mars Colony.


Speed of Simulation

1. Time Ratio : shows the ratio of the simulation time to the real time.

Note : you may speed up (or slow down) the simulation by dragging the handle to the right (or to the left).

e.g. if time ratio is 128x, then 1 real second equals 128 sim seconds.

2. Tick per Second (TPS) : shows how often the simulation updates the changes.

Note : TPS is essentially the same as "Frames Per Second" (FPS) in game-loop design.

e.g. if TPS/FPS is 25 cycles per sec, then for each second, the simulation is updated 25 times.

3. Simulation Uptime : displays how long the simulation has been up running.

4. Pause Button : click button or hit ESC to toggle between pause and resume the simulation.


Time and Season

1. Martian Time : the current Martian time in the format of Orbit-Month-Sol Millisol.

a. Martian Calendar : displays a Martian Calendar with the sol of the week/month highlighted.

Note : An orbit is a Martian Year. Time of a sol is in millisols (between 0 and 1000).

b. Martian Seasons : shows the current season on the Northern and Southern hemisphere of Mars.

c. Solar Longitude of Mars (L_s): represents the Mars-Sun angle, measured from the Northern Hemisphere spring equinox where L_s = 0. L_s = 90 thus corresponds to summer solstice, just as L_s = 180 marks the autumn equinox and L_s = 270 the winter solstice (all relative to the northern hemisphere). see [1] [2] for details and info on converting an Earth Date and time to Martian Date and Solar Longitude

2. Earth Time : shows the corresponding date and time on Earth in Universal Time (UT).


Design of the Mars Calendar

The Martian calendar used in this simulation is based on Shaun Moss's Areosynchronous Calendar.

The Areosynchronous Calendar is heavily derived from Tom Gangale's Darian Calendar [1] [2] [3].

The Martian year is referred to as an "orbit". It is 668.5921 Martian days ("Sols") long. In the calendar, about half of the orbits are 668 Sols long, and the other half are "leap-orbits" with 669 Sols. See the Areosynchronous Calendar for complete details on leap-orbits.

The month names are based on Frans Blok's Rotterdam System.

There are seven sols in a Martian week with the following week names:

  1. Heliosol
  2. Neriosol
  3. Libersol
  4. Terrasol
  5. Venusol
  6. Mercusol
  7. Jovisol

Every Martian month has 4 weeks. On months with 27 Sols, the last week only has six Sols and Jovisol is clipped. This allows the first Sol of every month and every orbit to be Heliosol.

The calendar uses metric time for Mars with the Sol broken up into the following common units:

decasol = 1/10 Sol = 2.46 Earth hours
centisol = 1/100 Sol = 14.8 Earth minutes
millisol = 1/1000 Sol = 1.48 Earth minutes

Martian time of the Sol is usually written in three-digit millisol format to three decimal points. ex. from "000.000" to "999.999".

The common timstamp format for Mars date/time is orbit-month-sol:millisols. ex. "17-Adir-03:523.234"

The time of the day is based on Bruce Mackenzie's Metric Time for Mars.
