Configuration Editor

The Configuration Editor allows you to configure what type of initial settlements you would like to start with in a new simulation you are going to create.

It displays a table of settlements soon to be created in the new simulation. The table is initially populated with a default set of settlements and allows users to customize the following characteristics of a settlement :

  1. Settlement Name
  2. Settlement Template
  3. Population
  4. # of Bot
  5. Sponsoring Agency
  6. Latitude
  7. Longitude

If a field is changed to an invalid value, a red error message appears below the table and the 'Start' button will be disabled until it is fixed.

The 'Add' button will add a new settlement with a random configuration to the table.

The 'Remove' button will remove any selected settlements from the table.

The 'Undo' button will clear all user changes and reset the table to holding only the two default settlements.

The 'Start' button will close the dialog, create and start a new simulation with the configured settlements.

The 'Crew Editor' button allows you to edit the alpha crew roster.

Crew Editor

The Alpha Crew Editor presents the user a degree of freedom in customizing the profile of 4 settlers. Their characteristics were pre-configured in people.xml. You may choose their common settlement destination and alter the followings :

  1. Name
  2. Gender
  3. Job
  4. Sponsor
  5. Country
  6. MBTI Personality
