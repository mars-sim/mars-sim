Tabs

The Tab bar is located near the top of the main window, right below the title bar if mars-sim is in windows mode. User may switch to any one tab at any time.

3 Available Tabs :

  • Help tab
  • Map tab
  • Main tab

    • The Help tab houses the Help Browser for viewing the about page, tutorials, user guide and various mars-sim related sites. It's also a functional HTML5 compliant browser.

    The Map tab houses Settlement Map as well as the Mars Navigator Minimap.

    The Main tab is where 5 types of Unit Info Windows may be displayed :

    • Person
    • Bot
    • Vehicle
    • Settlement
    • Equipment

    Also, the Main tab houses the following Tool Windows :

    • Help Tool (F1): will jump to the Help Tab.
    • Search Tool (F2): search for a settler.
    • Time Tool (F3): display various simulation time and speed setting.
    • Monitor Tool (F4): the table where various types of settlement data are aggregated and displayed via graphs and charts using the Monitor Tool.
    • Mission Tool (F5): the interface for displaying, resourcing, planning and interacting with various type of Missions.
    • Science Tool (F6): the panel for providing insights into the ongoing and finished scientific studies and scientific achievement of the settlers.
    • Resupply Tool (F7): the one-stop shop for ordering supplies and parts from Earth for expanding existing settlements or staging new settlements on Mars.

    (See User Interface for details on each particular Tool)
