Resource : "Magnetite" (Solid)


1. Description :

  1. A rock mineral and one of the main iron ores, with the chemical formula Fe3O4. It is one of the oxides of iron, and is ferrimagnetic; it is attracted to a magnet and can be magnetized to become a permanent magnet itself.
