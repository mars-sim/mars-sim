Tutorial - Step 2: The Main Simulation Window

The Main Simulation Window is the large window which forms the base of the simulation. It has a reddish background texture, a menu bar, a tool bar and a bottom bar which lists open unit windows. It can also display a multitude of tool windows.

Upon start of a new simulation, you will see one tool window already open: the Mars Navigator. We'll cover that one in step 4.
